# SpringBoot ElasticSearchExample

Basic Example Using Rest-Client (Low & High Level) in SpringBoot Application


A simple Java Project to implement Elasticsearch Low-Level and High-Level Rest Client 


## Usage

mvn:
clean install spring-boot:run
